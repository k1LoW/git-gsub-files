# git-gsub-files

A Git subcommand to `git mv` file like gsub in a repository

## Usage

```sh
$ git gsub-files READ WRITE
$ git status
On branch master
Changes to be committed:
(use "git reset HEAD <file>..." to unstage)

        renamed:    README.md -> WRITEME.md
```

## Instration

    $ gem install git-gsub-files
